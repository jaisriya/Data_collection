# Python Data Manipulation & API Integration

This repository contains a Google Colab project demonstrating foundational data engineering and analytics skills using Python. It covers open-source data manipulation, REST API handling, relational database merging, and replicating traditional Excel operations using Pandas.

## Features
* **Data Wrangling:** Loading, inspecting, and manipulating open-source datasets (Titanic) with Pandas, including sorting, central tendency, and standard deviation calculations.
* **REST API Integration:** Fetching, parsing, and structuring JSON data from the OpenWeatherMap API into tabular formats for multiple cities.
* **Relational Data Merging:** Simulating primary and secondary foreign key joins across generated datasets (100-row and 1,200-row DataFrames) using Pandas `merge`.
* **Excel to Python Translation:** Replicating VLOOKUP functionality, conditional filtering, and automated file I/O operations (CSV export/import).
